# 🏥 Health-Nexus

Health Nexus is a **C++ DSA-based Hospital Management System** that integrates core hospital operations into a single unified platform.  
It simulates real-world hospital workflows using efficient **Data Structures and Algorithms** concepts.

---

## 🚀 Features

### 👤 Patient Management
- Add new patients  
- Search, update, and view patient records  
- Maintain complete patient history  

### 🛏️ Bed & Ward Management
- Allocate hospital beds  
- Track available and occupied beds  
- Ward-wise organization system  

### 📅 Appointment System
- Schedule doctor appointments  
- Prevent time conflicts  
- View upcoming appointments  

### 💊 Pharmacy Management
- Add, update, and delete medicines  
- Manage inventory stock  
- Track medicine availability  

### 💰 Billing System
- Generate patient bills  
- Include treatment, medicines, and stay charges  
- Store billing records  

---

## 🧠 Core Concepts Used (DSA)

- Arrays  
- Linked Lists  
- Stacks & Queues  
- Searching & Sorting Algorithms  
- Hashing for fast lookup  
- File Handling (persistent storage)  
- Modular programming in C++  

---

## 🛠️ Technologies Used

- C++  
- STL (Standard Template Library)  
- File Handling (`fstream`)  
- Console-Based Interface
